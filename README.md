# GoOS
GoOS is a small scaled opearating system designed by Goplex Studios.

For the version support system, see SUPPORT.MD

# Features
 - It can start up with no errors.
 - Can run on both Legacy and modern hardware
 - Like MS DOS but with colors
 - Has 2 games (TEXTADVENTURES and SNAKE)
 - Networking (In Testing)
 - Active Development
 - Almost 2 years of work
 - Has GoOS Core as a backend
